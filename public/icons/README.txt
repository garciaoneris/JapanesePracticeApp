Place the following PNG files here before the first production build:

  icon-192.png             192x192
  icon-512.png             512x512
  icon-maskable-512.png    512x512, with safe zone (80% center)

Any simple generator works (e.g. https://realfavicongenerator.net, or a single
SVG piped through ImageMagick). Without these the PWA will still install but
Safari will show a generic icon.
